# CODE OF CONDUCT

## The One Rule

**Don't be useless.**

Everything else flows from that.

---

## What This Place Is

This is a research archive for the Rohonc Codex decipherment. It exists to preserve findings, enable verification, and let others build on the work.

It is not a community center. It is not a safe space. It is not a forum for people to process their feelings about linguistics.

**The work is the point. Everything else is noise.**

---

## What Is Expected

### Bring Signal, Not Noise

- If you have something to add, add it. Evidence, corrections, tools, analysis.
- If you have a question, ask it directly. No preamble. No performative context.
- If you disagree, say why with data. "I disagree because X" is the format.

### Own Your Words

- Stand behind what you post. If you say something, you believe it and can defend it.
- Anonymous handles are fine. Standing behind your output is not optional.
- If you were wrong, say so. Correct it. Move on.

### Respect the Work's Scope

- This project is about the Rohonc Codex specifically.
- Stay on topic. Off-topic agendas are removed.
- Don't try to use this space to fight proxy wars about other things.

---

## What Gets You Removed

There is no complex matrix of violations. The following behavior ends your access:

1. **Deliberate sabotage** – corrupting data, submitting known-false information, poisoning the lexicon
2. **Bad faith engagement** – pretending to contribute while actually trying to discredit, delay, or disrupt
3. **Harassment of contributors** – following people across threads, targeting individuals
4. **Spamming** – repeated low-quality submissions to manufacture visibility

Note: Being wrong is not a violation. Being annoying is not a violation. Being bad at linguistics is not a violation. Being actively destructive is.

---

## Enforcement

The Operator handles enforcement. There is no appeals board.

If you're removed, you're removed. If you think it was a mistake, you can send one message explaining why. If it was a mistake, it gets reversed. If it wasn't, it doesn't.

Redemption is always possible through merit. See the [main README](README.md#removal-clause-redemption-via-merit) for the full clause.

---

## What This Code of Conduct Is Not

- A list of protected characteristics
- A commitment to "center" any group's experience
- A promise that you will never feel uncomfortable
- A guarantee that your contributions will be accepted
- A mechanism for ideological enforcement

If you came here looking for any of those things, this is the wrong project.

---

## On Respect

Respect here means: **take the work seriously, take accuracy seriously, take other contributors' time seriously.**

It does not mean: validate everyone's interpretation, defer to seniority, or soften criticism to protect feelings.

Blunt, evidence-backed criticism of ideas is not disrespect. Personal attacks are. Know the difference.

---

## Summary

Show up. Do the work. Be honest. Don't wreck things.

That's it.
